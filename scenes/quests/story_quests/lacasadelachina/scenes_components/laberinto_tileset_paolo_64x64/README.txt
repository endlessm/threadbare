PAQUETE LABERINTO + TILESET 64x64

Contenido:
- mapa_laberinto_referencia.png: mapa visual del laberinto con 4 runas.
- tileset_atlas_referencia.png: atlas visual de piezas para reconstruirlo.

Configuración recomendada en Godot:
- TileMap / TileSet con Tile Size: 64 x 64 px.
- Margin: 0.
- Separation: 0.
- Usa el mapa como referencia visual y el atlas como base de piezas.

Nota:
Las imágenes son una referencia renderizada. Para un TileSet perfecto 1:1, conviene recortar/redibujar cada tile individual en 64x64 en Piskel/LibreSprite siguiendo este atlas.
